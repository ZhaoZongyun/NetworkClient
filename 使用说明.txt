1. 将 proto 文件放入 Network/Protobuf/ProtoFiles 目录中
    （1）该目录下已有的 proto 文件为示例，可删除

2. 运行 Network/Protobuf/Tool 目录中的 protoGen.bat 文件，会根据 proto 文件生成对应的 c# 代码
    （1）在 window 目录下双击允许，直接在 Unity 中双击无效

3. 运行 Unity 菜单中的 proto/生成NetModule代码
    （1）会生成 NetModuleGen.cs 和 NetMessageId.cs 两个脚本
    （2）NetModuleGen.cs 处理网络消息收发，NetMessageId.cs 为消息id枚举

4. 需要收发消息时，按如下示例使用，或参考 TesNet.cs 脚本

    public void Test()
    {
        //发送网络消息
        NetModule.Instance.ReqCreateCharacter("张三", 1);

        //接收网络消息
        NetModule.Instance.AddNetMessageEvent(NetMessageId.ResCreateCharacter, ResCreateCharacter);
    }

    private void ResCreateCharacter(IMessage obj)
    {
        ResCreateCharacter data = obj as ResCreateCharacter;
        if (data.ResCode == 1)
            Log.Info("昵称不合法");
        else if (data.ResCode == 2)
            Log.Info("昵称已存在");
        else if (data.ResCode == 0)
            Log.Info("创角成功");
    }
